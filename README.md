# 中草药智能显微图库平台介绍

> 作者：csh
>
> Email: 2502781981@qq.com

## 项目介绍：

基于 Spring Boot + Redis + COS + AI + WebSocket 的中草药智能显微图库平台。

分为公共图库、私有图库和团队共享图库三大模块。公共图库支持用户上传、检索与下载；私有图库仅个人访问；团队图库支持邀请成员协同编辑。管理员可进行图片上传、审核与管理。

1) 公共图库页：所有用户都能访问并下载公共图库中的图片素材，可通过搜索快速找到想要的图片。

![](https://csh-pictures-1317136289.cos.ap-guangzhou.myqcloud.com/github-md/Snipaste_2025-11-17_21-02-22.jpg)

2. 管理员可上传、审核和管理图片。

![](https://csh-pictures-1317136289.cos.ap-guangzhou.myqcloud.com/github-md/2.jpg)

3. 个人空间，可将显微图片上传至私有空间，用作个人显微图库。

![](https://csh-pictures-1317136289.cos.ap-guangzhou.myqcloud.com/github-md/3.jpg)

4. 以图搜图，可按照已有图片到互联网上搜索类似图片。

![](https://csh-pictures-1317136289.cos.ap-guangzhou.myqcloud.com/github-md/4.jpg)

5. 可对已有空间进行空间分析，可显示存储空间用量、图片数量、标签分析、图片大小分析等。

![](https://csh-pictures-1317136289.cos.ap-guangzhou.myqcloud.com/github-md/5.jpg)

6. 团队协作，可邀请用户共同访问、管理空间内容

![](https://csh-pictures-1317136289.cos.ap-guangzhou.myqcloud.com/github-md/6.jpg)
